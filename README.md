## Assumptions
The model uses base-case assumptions based on industry benchmarks and historical trends from a group of similar companies. Median values from peer data were used to estimate "Comps" (Comparable Firm Analysis). An intelligent set of financial statement projection assumptions were used, which was simplified to focus on building and understanding the DCF framework. Refer "Assumptions" and "Comps" sheet tabs.
