# Company-Valuation-DCF-and-Comps
## Project Overview
Performed a valuation of Cardinal FT Company as of April 1, 2018. The company operates through two distinct business segments: manufacturing and services. The model forecasts the company’s financial performance over a five-year period on a quarterly basis through the end of the year 2022.
## Project Objective
- Estimate the enterprise value and per share price of Cardinal FT Company as of April 1, 2018.
- Build consolidated pro-forma financial statements, including the income statement, balance sheet, and cash flow statement.
- Apply the income approach (DCF valuation) to find out: a. Terminal Value using perpetual growth method b. Terminal Value using terminal multiple method.
- Perform a comparable company analysis by applying appropriate multiples (market value ratios) from similar publicly listed firms.
## Assumptions
The model uses base-case assumptions based on industry benchmarks and historical trends from a group of similar companies. Median values from peer data were used to estimate "Comps" (Comparable Firm Analysis). An intelligent set of financial statement projection assumptions were used, which was simplified to focus on building and understanding the DCF framework. Refer "Assumptions" and "Comps" sheet tabs.
